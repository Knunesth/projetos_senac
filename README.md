# 📚 Clube do Livro — Landing Page React

Uma landing page moderna desenvolvida em **React** para apresentar um serviço fictício de **clube de livros por assinatura**.

O projeto foi criado com foco em **UX (User Experience)**, **responsividade** e **componentização**, utilizando conceitos de **Atomic Design**.

# 🎨 UX Decisions

O projeto foi desenvolvido seguindo algumas boas práticas de experiência do usuário:

* **Mobile First**
* Layout totalmente **responsivo**
* **Scroll suave** entre seções
* **Menu hambúrguer** no mobile
* **CTAs claros** para conversão
* Cards visuais para facilitar leitura
